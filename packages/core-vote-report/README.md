## Installation

```bash
yarn add @arkecosystem/core-vote-report
```

## Configuration

```js
module.exports = {
  host: process.env.ARK_VOTE_REPORT_HOST || '0.0.0.0',
  port: process.env.ARK_VOTE_REPORT_PORT || 4006
}
```
