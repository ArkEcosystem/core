# ARK Core - Transaction Pool Sequalize

## Installation

```bash
yarn add @arkecosystem/core-transaction-pool-sequalize
```

## Configuration

### Defaults

```js
module.exports = {
  enabled: true,
  key: 'ark',
  maxTransactionsPerSender: 100,
  allowedSenders: []
}
```

## Security
