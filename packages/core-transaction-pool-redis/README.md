# ARK Core - Transaction Pool Redis

## Installation

```bash
yarn add @arkecosystem/core-transaction-pool-redis
```

## Configuration

### Defaults

```js
module.exports = {
  enabled: true,
  key: 'ark',
  maxTransactionsPerSender: 100,
  allowedSenders: [],
  redis: {
    host: process.env.ARK_REDIS_HOST || 'localhost',
    port: process.env.ARK_REDIS_PORT || 6379
  }
}
```
