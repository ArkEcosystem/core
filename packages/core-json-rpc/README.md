## Installation

```bash
yarn add @arkecosystem/core-json-rpc
```

## Configuration

```js
module.exports = {
  enabled: !process.env.ARK_JSON_RPC_DISABLED,
  host: process.env.ARK_JSON_RPC_HOST || '0.0.0.0',
  port: process.env.ARK_JSON_RPC_PORT || 8080,
  allowRemote: true,
  whitelist: ['127.0.0.1', '::ffff:127.0.0.1', '192.168.*']
}
```
