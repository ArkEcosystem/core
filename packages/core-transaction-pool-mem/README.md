# ARK Core - Transaction Pool (Mem)

## Installation

```bash
yarn add @arkecosystem/core-transaction-pool-mem
```

## Configuration

### Defaults

```js
module.exports = {
  maxTransactionsPerSender: 100,
  allowedSenders: []
}
```

## Security
